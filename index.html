⚙️ Setup Koneksi Google Sheets

Ikuti langkah berikut untuk menghubungkan web app ini dengan Google Sheets. Cukup dilakukan sekali saja.

Buat Google Sheets baru

Buka sheets.new → buat spreadsheet baru dengan nama "Manajemen Sawah". Buat 3 sheet tab: PANEN_LOG, ASET, JADWAL.

Salin Spreadsheet ID dari URL (bagian antara /d/ dan /edit):

Buat Google Cloud Project & API Key

  1. Buka console.cloud.google.com
  2. Buat project baru → New Project
  3. Aktifkan Google Sheets API dan Google Drive API
  4. Buat OAuth 2.0 Client ID (tipe: Web application)
  5. Tambahkan Authorized JavaScript origin: URL tempat app ini dihosting (atau http://localhost untuk testing)
  6. Copy Client ID:

Inisialisasi Header Kolom di Google Sheets

Setelah mengisi Spreadsheet ID dan Client ID di atas, klik tombol ini untuk otomatis mengisi header kolom di Google Sheets Anda:

Import data historis dari Excel

Buka file Management_Sawah_v2.xlsx yang sudah dibuat sebelumnya → copy data dari sheet 📈 PANEN LOG → paste ke sheet PANEN_LOG di Google Sheets Anda mulai baris ke-2.

🗂️ Pilih Sheet & Format Data
💡 Jika pakai file Management_Sawah_v2.xlsx yang sudah ada di Google Drive, pilih opsi "File Excel lama"
